#### SYNOPSIS

 [[-Password] <SecureString>] [-UseSsl] [-AlwaysSend] [-SendOnError] [-SendOnHost] [-SendOnOutput]
 [-SendOnVerbose] [-SendOnWarning] [-SendOnDebug] [-SendOnInformation] [-AttachLogfile] [-KeepLogging]
